# FashionMNIST Image Classification with PyTorch

A Convolutional Neural Network (CNN) built with PyTorch to classify images from the FashionMNIST dataset. This project demonstrates the complete deep learning pipeline, from data loading and preprocessing to model training and evaluation.

## Features

- CNN implemented using PyTorch
- FashionMNIST dataset from Torchvision
- Data preprocessing with DataLoader
- Model training and evaluation
- Test accuracy of **83.92%**


## Tech Stack

- Python
- PyTorch
- Torchvision
- Google Colab
- Matplotlib

## Dataset

The project uses the **FashionMNIST** dataset containing **70,000 grayscale images (28×28)** across **10 clothing categories**.

## Results

- **Train Loss:** 0.4518
- **Test Accuracy:** 83.92%
